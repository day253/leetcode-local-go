# Data Science Leetcode 精简版

## 划分说明

此表是针对 Data Science 这个职位，对Leetcode前400题进行精简，划分精简规则如下：

- 删除DS不常考，面试低频出现题目
- 删除SDE考，过难题目
- 删除高级数据结构算法，保留基础简单题目
- 保留基础练手题目，提高代码能力

所有题目我们尽量保证客观公正，只是按大概率删除，目的是为了减轻DS的刷题负担。

适用人群：Data Science 职位相关人员
